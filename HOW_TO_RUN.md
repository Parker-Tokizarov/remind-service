# Инструкция по запуску RemindService

## Шаг 1: Установка зависимостей

```bash
cd remind-service
npm install
```

## Шаг 2: Настройка Supabase

### 2.1 Создайте проект

1. Перейдите на [supabase.com](https://supabase.com)
2. Войдите через GitHub или создайте аккаунт
3. Нажмите "New Project"
4. Заполните данные проекта и создайте его

### 2.2 Получите ключи API

1. В проекте перейдите в **Settings** → **API**
2. Скопируйте:
   - **Project URL** → `NEXT_PUBLIC_SUPABASE_URL`
   - **anon public** → `NEXT_PUBLIC_SUPABASE_ANON_KEY`

### 2.3 Создайте таблицы

1. Перейдите в **SQL Editor** (в левом меню)
2. Нажмите **New Query**
3. Скопируйте содержимое файла `schema.sql`
4. Вставьте в редактор и нажмите **Run**

### 2.4 Включите Email аутентификацию

1. Перейдите в **Authentication** → **Providers**
2. Убедитесь, что **Email** включен
3. Для тестирования можно отключить подтверждение email:
   - **Authentication** → **Settings** → **Email Auth**
   - Отключите "Confirm email"

## Шаг 3: Настройка переменных окружения

Создайте файл `.env.local` в корне проекта:

```bash
NEXT_PUBLIC_SUPABASE_URL=https://xxxxxxxxxxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

## Шаг 4: Запуск приложения

### Режим разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### Продакшен сборка

```bash
npm run build
npm start
```

## Шаг 5: Регистрация первого пользователя

1. Откройте приложение в браузере
2. Нажмите "Зарегистрироваться"
3. Введите email, пароль и имя
4. После регистрации вы попадете на дашборд

## Шаг 6: Настройка Telegram (опционально)

### Создание бота

1. Откройте Telegram, найдите @BotFather
2. Отправьте команду `/newbot`
3. Введите имя и username бота
4. Скопируйте полученный токен

### Получение Chat ID

1. Найдите своего бота в Telegram, нажмите Start
2. Найдите бота @userinfobot
3. Отправьте ему любое сообщение
4. Скопируйте ваш Chat ID

### Настройка в приложении

1. Перейдите в **Настройки** в приложении
2. Вставьте токен бота и Chat ID
3. Нажмите "Тест" для проверки

## 🐛 Решение проблем

### Ошибка "Failed to fetch"

Проверьте правильность URL и ключей Supabase в `.env.local`

### Ошибка аутентификации

Убедитесь, что email подтвержден (или отключите подтверждение в настройках Supabase)

### Ошибка при отправке Telegram

Проверьте токен бота и Chat ID. Убедитесь, что вы писали боту после создания.

## 🚀 Деплой на Vercel

1. Запушьте код на GitHub
2. Перейдите на [vercel.com](https://vercel.com)
3. Импортируйте репозиторий
4. Добавьте переменные окружения в настройках проекта
5. Нажмите Deploy

После деплоя приложение будет доступно по вашему домену Vercel.
